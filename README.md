# Mess Feedback Analysis System

A web-based feedback system developed using Flask and C++ for collecting and analyzing student mess reviews.

## Features
- Submit food and mess-related feedback through a simple web interface.
- Analyze feedback using keyword extraction and sentiment scoring.
- Identify frequently appreciated and criticized food items.
- Store all submitted reviews for future analysis.

## Technologies Used
- Python (Flask)
- C++
- HTML/CSS

## Usage
Students can submit feedback through the web form, while administrators can review collected feedback and generated insights.
